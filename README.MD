# Lotus

#### It's is an eCommerce website simple build with Python and use framework Flask

## Demo clip
- [Youtube](https://www.youtube.com/watch?v=-WSOdjO9KRM)


## Features
- Views all product
- Add product to cart
- Checkout
- Review product

## How to install 

#### Install all packages
```
$ pip install -r requirement.txt
```

## How to run
```
$ python app.py 
```
